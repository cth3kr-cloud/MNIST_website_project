# 인터페이스 명세서 (Interface Specification)

## 프로젝트 개요

프로젝트명: NumPy 기반 손글씨 숫자 인식 AI 웹 서비스

### 🎯 목표:

* **NumPy만 사용하여 인공신경망 구현**
* **사용자가 웹에서 숫자를 입력하면 AI가 판별**
* **웹 서비스 형태로 제공**

---

## 전체 시스템 구조

```
사용자

↓

프론트엔드 (HTML, CSS, JavaScript)

↓

백엔드 (FastAPI)

↓

AI 모델 (NumPy)

↓

백엔드

↓

프론트엔드

↓

사용자
```
